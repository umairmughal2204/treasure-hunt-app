🔍 Treasure Hunt Web App

Discover clues, solve puzzles, and compete in real-world hunts!

🌟 What is this?

This mobile web app allows users to participate in GPS-based treasure hunts powered by the CodeCyprus API. Solve questions at real locations, earn points, and compete!

📲 Why Use This App?

  • Location-aware question validation
  • Live score tracking
  • Multiple-choice, text, numeric & boolean question types
